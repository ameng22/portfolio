# Amen George - Portfolio

Professional portfolio website for Amen George, a Software Developer specializing in cloud architecture and backend systems.

## Project Info

This project is a modern, responsive portfolio website built to showcase skills, projects, and experience.

## Tech Stack

- **Framework:** React with Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Icons:** Lucide React

## Getting Started

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone <repository-url>
    cd portfolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  **Build for production**
    ```bash
    npm run build
    ```
